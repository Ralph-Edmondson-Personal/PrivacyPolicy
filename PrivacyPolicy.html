Privacy Policy for StopIn.

Last Updated: November 2025

1. Overview

StopIn. ("the App") is an event discovery and social platform that allows users to create, explore, and interact with house events and gatherings. This Privacy Policy outlines how we collect, store, and use your data.

StopIn. uses Firebase services provided by Google for authentication, database storage, and secure hosting. By using the App, you agree to the practices described in this policy.

2. Information We Collect

2.1 Information You Provide

We collect information that you manually input, including:

  • Name
  • Email address
  • Event details (title, description, visibility, time range)
  • Profile information
  • Events you create or join
  • Images uploaded to events

2.2 Automatically Collected Information

Firebase may automatically collect:

  • Device identifiers
  • Crash logs
  • Performance data
  • General usage analytics

This data is used to improve stability and performance.

3. How We Use Your Information

Your information is used only to support app functionality, including:

  • Displaying events and user-generated content within the app
  • Authenticating users securely
  • Allowing users to host and join events
  • Showing event ratings and participants
  • Improving overall app performance

We do not sell or share your personal information with advertisers or unrelated third parties.

4. Data Storage

Your data is stored securely using Firebase Firestore and Firebase Storage. This includes:

  • User accounts
  • Created events
  • Event images
  • QR code identifiers for approved attendees

Firebase stores information on secure Google servers following industry-standard protections.

5. Data Sharing

Your information is shared only when necessary:

  • With other users (basic profile info, created events)
  • With Firebase to provide app functionality

Firebase Privacy Policy: https://firebase.google.com/support/privacy

6. Data Security

We rely on Firebase’s built-in security features such as encrypted communication, authentication protection, and Firestore Security Rules to ensure your data remains safe.
